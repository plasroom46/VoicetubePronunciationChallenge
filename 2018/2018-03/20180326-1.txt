1. 
a. meditate (v.)  冥想
b. stretch (v.)  拉筋
  -> I always stretch before I work out, so that I don't get hurt.
    我在訓練前都會先拉筋，以免受傷。
c. yoga (n.)  瑜珈
  -> Grace does yoga twice a week, her favorite yoga pose is child's pose.
    Grace每週做兩次瑜伽，她最喜歡的瑜伽姿勢是嬰孩式。
2. 
a. chore (n.)  家事
  * 除了一般的家事，也可以用來形容麻煩或不想做的事情。
  -> I find driving to work a real chore.
    我覺得開車去上班真的很煩。
b. errand (n.)  差事
  * 通常需要外出進行
  * 可以用於有很多繁雜的私事要處理，而又不想詳細說明時
  -> run an errand 去某處處理某件事或幫別人完成某事
  -> I need to run some errands this afternoon.
    我下午需要處理一些事情。
c. duty (n.)  責任; 職責
  -> I work at a cafe. My duties include making coffee and serving customers.
    我在咖啡廳工作，我的職責包括製作咖啡及服務客人。
3. 
a. productivity (n.)  生產力
  -> Making a schedule can help you increase your productivity.
    製作一個行程表能夠上幫助你提升你的生產力。
  -> high productivity/ good productivity 高生產力
b. productive (adj.)  生產的; 有成效的
  -> I'm more productive in the morning.
    我在早上時做事效率比較高。
  -> We had a very productive meeting this afternoon.
    我們下午的會議進行得很有效率。
c. efficient (adj.)  效率高的
  =productive = wasting little effort
  -> Mark is our most efficient worker.
    Mark是我們辦事最有效率的員工。
  -> The most efficient way to clean a floor is by using a vacuum.
    清潔地板最有效率的方式是使用吸塵器。
4. 
a. machine (n.)  機器
  * 若形容人，表示某人很擅長某件事
  -> He is a machine.
    他做什麼事都很厲害。
  -> Tiger Woods is a golfing machine.
    老虎伍茲很擅長打高爾夫。
b. robot (n.)  機器人
  * 若形容人，為貶義，代表他沒有什麼感情、不好相處
